📚 How to Import Bookmarks

Follow these step-by-step guides to import your bookmarks into your browser:

🌐 Google Chrome

1
Open Chrome Menu

Click the three dots (⋮) in the top-right corner of Chrome

2
Go to Bookmarks

Hover over "Bookmarks" and select "Import bookmarks and settings"

3
Select File

Choose "Bookmarks HTML file" and click "Choose File"

4
Import

Select your downloaded bookmarks.html file and click "Import"

🌐 Microsoft Edge

1
Open Edge Menu

Click the three dots (⋯) in the top-right corner of Edge

2
Go to Favorites

Select "Favorites" → "Import favorites"

3
Choose Source

Select "Import from file" and choose "Favorites or bookmarks HTML file"

4
Import

Browse and select your bookmarks.html file, then click "Import"

🦊 Mozilla Firefox

1
Open Library

Click the library button (📚) in the toolbar or press Ctrl+Shift+O

2
Import & Backup

Click "Import and Backup" → "Import Bookmarks from HTML"

3
Select File

Choose your downloaded bookmarks.html file

4
Complete

Your bookmarks will be imported automatically

🧭 Apple Safari

1
Open File Menu

Click "File" in the Safari menu bar

2
Import Bookmarks

Select "Import From" → "Bookmarks HTML File"

3
Choose File

Navigate to and select your bookmarks.html file

4
Import

Click "Import" to add your bookmarks to Safari

🦁 Brave Browser

1
Open Brave Menu

Click the three lines (☰) in the top-right corner

2
Bookmarks Manager

Go to "Bookmarks" → "Bookmark Manager"

3
Import

Click the three dots (⋮) → "Import bookmarks"

4
Select File

Choose "Bookmarks HTML file" and select your downloaded file
